Cybersecurity Project

Drone Q-Kavach

Drone Q-Kavach is my quantum-resistant drone communication project, designed around secure telemetry, authenticated commands, encrypted video, and replay protection.

Role
Security architecture and prototype planning
Stack
Kyber, Dilithium, AES-256-GCM, secure communication design
Focus
Drone communication, telemetry protection, secure live video

What I tried to solve

Drones can depend on wireless links for commands, telemetry, and video. If that link is weak, an attacker can try replay attacks, command tampering, or traffic interception. I designed this project around stronger identity, encrypted telemetry, and post-quantum-ready key exchange.

System ideas

  • Kyber-based session key exchange for quantum-resistant communication.
  • Dilithium signatures for authenticating commands and messages.
  • AES-256-GCM for encrypted telemetry and secure payload handling.
  • Replay protection and mutual authentication between drone and controller.
  • Secure live video streaming as a future-ready extension.
