Privacy Product

SecureShare+ / Secure Print

Secure Print+ is a privacy-focused file sharing and printing workflow for public systems such as cyber cafes, where files should not remain visible or stored after use.

Role
Product idea, Flutter/Firebase implementation
Stack
Flutter, Firebase Auth, Firestore rules, secure links
Focus
One-time access, auto-expiry, privacy-first printing

The problem

People often print documents from public computers, but those systems can retain files or expose sensitive content. I built this project around the idea that the public machine should not need permanent access to the document.

What matters technically

  • Auto-expiring secure links for temporary access.
  • Firebase Authentication and Firestore Security Rules.
  • Client-side access workflow designed around privacy.
  • Automated printing flow that reduces manual file exposure.
  • Zero-knowledge style product thinking for public systems.
